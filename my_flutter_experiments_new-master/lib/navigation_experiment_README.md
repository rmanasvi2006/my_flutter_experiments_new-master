# Navigation Experiment

This experiment explores navigation in Flutter apps, including moving between screens and passing data.

## Overview
- Demonstrates navigation using `Navigator.push` and `Navigator.pop`
- Shows how to pass arguments between screens
- Explains how to use named routes

## Key Concepts
- **Navigator:** Managing app navigation stack
- **Routes:** Defining and using named routes
- **Passing Data:** Sending information between screens

## How to Run
1. Open this project in your IDE.
2. Run the app using `flutter run`.
3. Navigate to the Navigation Experiment screen.

## Example Output
- Buttons to move between screens
- Data displayed on second screen after navigation

## Purpose
- Learn navigation fundamentals in Flutter
- Practice passing data and using routes
