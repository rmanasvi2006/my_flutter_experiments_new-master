# Responsive Layout Experiment

This experiment demonstrates how to build responsive UIs in Flutter that adapt to different screen sizes and orientations.

## Overview
- Uses `MediaQuery` and layout widgets to adjust UI
- Shows how to build flexible layouts for mobile, tablet, and desktop
- Explains best practices for responsive design in Flutter

## Key Concepts
- **MediaQuery:** Accessing device size and orientation
- **Flexible Widgets:** Using `Expanded`, `Flexible`, and `LayoutBuilder`
- **Adaptive UI:** Changing layout based on screen size

## How to Run
1. Open this project in your IDE.
2. Run the app using `flutter run`.
3. Navigate to the Responsive Layout Experiment screen.

## Example Output
- UI that changes layout for portrait/landscape
- Widgets that resize and reposition based on device

## Purpose
- Learn to build apps that look good on any device
- Practice responsive design techniques
