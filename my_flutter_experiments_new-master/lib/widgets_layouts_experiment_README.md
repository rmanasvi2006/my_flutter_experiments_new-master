# Widgets & Layouts Experiment

This experiment showcases basic Flutter widgets and layout techniques. It is designed to help you understand how to arrange and style widgets in a Flutter app.

## Overview
- Demonstrates common widgets: `Text`, `Container`, `Row`, `Column`, `ListView`, etc.
- Shows how to use padding, alignment, and colors
- Explains how to build reusable widget functions

## Key Concepts
- **Widgets:** Building UI with Flutter's widget tree
- **Layouts:** Using `Row`, `Column`, and `ListView` for arrangement
- **Styling:** Applying colors, padding, and text styles
- **Reusable Widgets:** Creating helper functions for repeated UI elements

## How to Run
1. Open this project in your IDE.
2. Run the app using `flutter run`.
3. Navigate to the Widgets & Layouts Experiment screen.

## Example Output
- Colored boxes arranged in rows and columns
- Styled text and containers
- Scrollable lists of widgets

## Purpose
- Learn the basics of Flutter UI construction
- Practice layout and styling
- Build reusable widget functions
