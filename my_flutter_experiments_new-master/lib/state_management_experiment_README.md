# State Management Experiment

This experiment demonstrates state management in Flutter using Provider and ChangeNotifier.

## Overview
- Uses Provider for dependency injection
- Implements ChangeNotifier for reactive state
- Shows how to update UI based on state changes

## Key Concepts
- **Provider:** Managing app-wide state
- **ChangeNotifier:** Notifying listeners of state changes
- **Reactive UI:** Updating widgets when state changes

## How to Run
1. Open this project in your IDE.
2. Run the app using `flutter run`.
3. Navigate to the State Management Experiment screen.

## Example Output
- Counter that updates when a button is pressed
- UI reacts to state changes

## Purpose
- Learn state management best practices
- Practice using Provider and ChangeNotifier
